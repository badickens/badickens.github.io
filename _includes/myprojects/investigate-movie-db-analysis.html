Introduction¶

This report will investigate The Movie Database (TMDb) dataset. This dataset contains information of ~10,000 movies.

Dataset includes the following properties:

        id, imdb_id, popularity, budget, revenue, original_title, cast, homepage, director, tagline, keywords,
        overview, runtime, genres, production_companies, release_date, vote_count, vote_average, release_year, 
        budget_adj, revenue_adj

This investigation will focus on the popularity of the various genres, directors, actors, and movie revenues.

Over the course of this report the following questions will be explored:

  • What genres are most popular?
  • Which directors are associated with movies with high revenues?
  • Who are the most featured (popular) actors?

Data Cleaning:¶

However, cast, director, and genres have missing data. Since they are unique string data for each movie referenced, there is not a general default string that can be used in their place and allow for useful analysis.¶

A solution is to remove the affected rows from this dataset. This will still allow sufficient remaining rows for analysis.¶

In addition, the data within the genres and cast columns need restructuring. Each string contains multiple values separated by the pipe '|' character. Each will be converted to Python lists to allow easier access to each value.¶

The following sections will detail the data cleaning steps required.¶

Conclusions¶

Analysis of The Movie Database (TMDb) have resulted in the following findings:¶

This investigation focused on the most detailed attributes available in the dataset - actors, directors, and genres that could still gleam potentially interesting information. Note: No other sources were used to verify or suppliment the data from the TMDb dataset.

There were some limitations on the scope of this analysis during the exploration of the dataset. This included missing actual revenues values, and missing production company references. Production company information was not included among the detailed attributes since there was too much missing to deem it useful for comparison. Overall, half of the information per movie was for those released between 2006 and 2015.

The following cursory observations were made during this investigation:

  • The predominent genre description for movies released from 1960 through 2015 within this dataset was Drama, followed by Comedy.
  • 2156 movies listed had above average revenue ( >= ~40.3 Million )
  • Half of the movies with above average revenues made more than ~114 Million
  • Highest revenue was ~2.8 Billion (Avatar), directed by James Cameron
  • Among the ~10,700 directors, Steven Spielberg was listed as having directed the most movies (25) among those with high revenues. He was followed by Clint Eastwood and Ridley Scott.
  • Regarding the ~18,000 unique actors/cast listed, some general findings were:
    • Harrison Ford (41 movies, ~8.9B), Tom Cruise (37 movies, ~8.8B), and Tom Hanks (46 movies, ~8.4B) were the top three male actors with the highest combined revenues of movies they each starred in. All three summing to ~26B.
    • Cameron Diaz (38 movies, ~7B), Scarlette Johansson (32 movies, ~6B), and Sandra Bullock (34 movies, ~5.9B) were the top female actors with the highest combined revenues of movies each. All three summing to ~19B.
    • The top 3 male actors with the most movies listed were Robert De Niro (72 movies, ~5.1B), Samuel L. Jackson (71 movies, ~7.1B), and Bruce Willis (62 movies, ~6.6B).
    • The top 3 female actors with the most movies were Susan Sarandon (47 movies, ~1.2B), Julianne Moore (46 movies, ~2.4B), and Meryl Streep (44 movies, ~3.2B).
